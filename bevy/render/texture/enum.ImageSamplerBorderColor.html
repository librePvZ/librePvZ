Enum bevy::render::texture::ImageSamplerBorderColor

source · 
pub enum ImageSamplerBorderColor {
    TransparentBlack,
    OpaqueBlack,
    OpaqueWhite,
    Zero,
}
Expand description

Color variation to use when the sampler addressing mode is ImageAddressMode::ClampToBorder.

This type mirrors wgpu::SamplerBorderColor.

Variants§

§

TransparentBlack

RGBA color [0, 0, 0, 0].

§

OpaqueBlack

RGBA color [0, 0, 0, 1].

§

OpaqueWhite

RGBA color [1, 1, 1, 1].

§

Zero

On the Metal wgpu backend, this is equivalent to Self::TransparentBlack for textures that have an alpha component, and equivalent to Self::OpaqueBlack for textures that do not have an alpha component. On other backends, this is equivalent to Self::TransparentBlack. Requires wgpu::Features::ADDRESS_MODE_CLAMP_TO_ZERO. Not supported on the web.
