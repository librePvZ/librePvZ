Module virtue::generate

source · [−]
Expand description

Code to help generate functions.

The structure is:

  • Generator
    • .impl_for(): ImplFor
      • .generate_fn(): FnBuilder
        • .body(|builder| { .. }): StreamBuilder

Afterwards, Generator::finish() must be called to take out the TokenStream produced.
